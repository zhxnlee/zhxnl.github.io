Cancer is a generic term for a large group of diseases that can affect any part of the body, and other terms used are malignant tumours and neoplams.

According to the World Health Organization (2022) and the lollipop chart on the left, Cancer is the second leading cause of death globally, accounting for an estimated 10 million deaths in 2019. In this page, we will be finding out more on the types of cancers, how much does it affect the global population, and its related causes.

The lollipop chart on the left shows the number of deaths for each cancer type over the years. The x-axis represents the total number of deaths for each Cancer Type in a particular year, while the y-axis represents all the Cancer Types. A filter slider can be used to visualise how the number of deaths for each Cancer Type changes, and a selection tool can be used to select the continents.
